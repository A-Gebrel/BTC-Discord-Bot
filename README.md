# BTC-Discord-Bot
Simple Discord Bot to check Transaction confirmations status (Unconfirmed / Confirmed with how many confirmations)


## Installation
Install rqeuirements ``pip install requests discord.py``

Add the Discord Bot Token in ``main.py``
